
Задание 2.3.2.TSK.A: отметить все выдачи с идентификаторами ≤50 как возвращённые.

___________________________________________________

MySQL	Решение: 
update `subscriptions`
set `sb_is_active` = 'N'
where `sb_id` <= 50
___________________________________________________

MS SQL	Решение:
update subscriptions
set sb_is_active = 'N'
where sb_id <= 50
___________________________________________________

Oracle	Решение:
update "subscriptions"
set "sb_is_active" = 'N'
where "sb_id" <= 50

Задание 2.2.2.TSK.B: для всех выдач, произведённых до 1-го января 2012-го года, уменьшить значение дня выдачи на 3.

___________________________________________________

MySQL	Решение: 
UPDATE `subscriptions` 
SET `sb_start` = DATE_ADD(`sb_start`, INTERVAL -3 day) 
WHERE `sb_start` < '2012-01-01';
___________________________________________________

MS SQL	Решение:
UPDATE subscriptions 
SET sb_start = DATEADD(day, -3, sb_start) 
WHERE sb_start < '2012-01-01';
___________________________________________________

Oracle	Решение:
UPDATE "subscriptions" 
SET "sb_start" = TO_DATE("sb_start") -3 
WHERE "sb_start" < TO_DATE('2012-01-01', 'YYYY-MM-DD');
