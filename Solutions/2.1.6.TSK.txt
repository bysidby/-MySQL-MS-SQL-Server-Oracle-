Задание 2.1.6.TSK.A: показать список авторов в обратном алфавитном порядке (т.е. «Я  А».
___________________________________________________

MySQL	Решение: 
SELECT a_name
FROM authors
order by a_name desc
___________________________________________________

MS SQL	Решение:
SELECT a_name
FROM authors
order by a_name desc
___________________________________________________

Oracle	Решение:
SELECT "a_name"
FROM "authors"
order by "a_name" desc
