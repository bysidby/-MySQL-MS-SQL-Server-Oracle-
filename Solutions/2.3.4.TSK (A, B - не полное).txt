
Задание 2.3.4.TSK.A: добавить в базу данных жанры «Политика», «Пси-хология», «История».

___________________________________________________

MySQL	Решение: 
REPLACE INTO `genres` (`g_id`, `g_name`) 
VALUES (NULL, 'Политика'), (NULL, 'Психология'), (NULL, 'История')
___________________________________________________

MS SQL	Решение:
MERGE INTO [genres] 
USING ( VALUES (N'Политика'), (N'Психология'), (N'История') ) AS [new_genres]([g_name]) ON [genres].[g_name] = [new_genres].[g_name] 
WHEN NOT MATCHED BY TARGET THEN 
INSERT ([g_name]) 
VALUES ([new_genres].[g_name]);
___________________________________________________

Oracle	Решение:
MERGE INTO "genres" 
USING (SELECT ( N'Политика' ) AS "g_name" 
FROM dual 
UNION 
SELECT ( N'Психология' ) AS "g_name" 
FROM dual 
UNION 
SELECT ( N'История' ) AS "g_name" 
FROM dual) "new_genres" ON ( "genres"."g_name" = "new_genres"."g_name" ) 
WHEN NOT MATCHED THEN 
  INSERT ("g_name") 
  VALUES ("new_genres"."g_name")
