
Задание 2.3.3.TSK.A: удалить информацию обо всех выдачах читателям книги с идентификатором 1.

___________________________________________________

MySQL	Решение: 
delete from `subscriptions`
where `sb_book` = 1
___________________________________________________

MS SQL	Решение:
delete from subscriptions
where sb_book = 1
___________________________________________________

Oracle	Решение:
delete from "subscriptions"
where "sb_book" = 1

Задание 2.2.3.TSK.B: удалить все книги, относящиеся к жанру «Клас-сика».

___________________________________________________

MySQL	Решение: 
delete from `books`
where `b_id` in (select `b_id` 
                from `m2m_books_genres`
                where `g_id` = 5)
___________________________________________________

MS SQL	Решение:
delete from books
where b_id in (select b_id 
                from m2m_books_genres
                where g_id = 5)
___________________________________________________

Oracle	Решение:
delete from "books"
where "b_id" in (select "b_id" 
                from "m2m_books_genres"
                where "g_id" = 5)

Задание 2.2.3.TSK.C: удалить информацию обо всех выдачах книг, про-изведённых после 20-го числа любого месяца любого года.

___________________________________________________

MySQL	Решение: 
delete from `subscriptions`
where EXTRACT(day FROM `sb_start`) between 21 and 31
___________________________________________________

MS SQL	Решение:
delete from subscriptions
where day([sb_start])  between 21 and 31
___________________________________________________

Oracle	Решение:
delete from "subscriptions"
where EXTRACT(day FROM "sb_start") between 21 and 31
